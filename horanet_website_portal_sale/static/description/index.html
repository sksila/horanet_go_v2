Factures en ligne, compte de dépôt, échéanciers

Payez vos échéances et factures en ligne avec compte de dépôt

Description

Ce module permet d'ajouter des échéanciers, de les payer en ligne, et de faire du compte de dépôt.
Par défaut ce module utilise Payzen comme mode de paiement.

Usage

Définissez des échéanciers en allant dans "Compabilité / ". Allez simplement dans votre portail pour régler vos échéances et mettre de l'argent sur votre compte.
