Utilisez le nom et le prénom pour l'inscription

Un outil pour améliorer le système d'inscription d'Odoo

Description

Ce module ajoute les fonctionnalités suivantes :

  • Utilisez désormais le nom et le prénom dès lors de l'inscription.
  • Ajout d'un champ de confirmation de l'adresse mail pour éviter les erreurs de saisie
  • L'utilisateur inscrit est alors inactif tant qu'il ne se sera pas connecté une fois.
  • Envoi un mail de bienvenue lors de la première connexion.

Usage

Activez l'enregistrement par le portail Odoo dans la configuration. Enregistrez-vous simplement par le portail.

Utilisation

Un bouton présent dans la vue formulaire des partenaires permet de créer un compte utilisateur rapidement

Le groupe d'accès au portail est celui défini dans le menu de configuration de la connexion
