Validez vos partenaires

Validez leur adresse ainsi que leur relations

Description

Ce module introduit un système de validation des partenaires. Il permet :

  • De valider l'adresse et les relations des partenaires directement depuis sa fiche et en lien avec les documents fournis

Usage

Dans la vue de validation des partenaires, deux boutons apparaissent pour valider leurs statut d'adresse et de relation.



Un wizard apparaît pour sélectionner le document justificatif et valider les relations ou l'adresse.
