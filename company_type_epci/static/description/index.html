Gérez les EPCI et communes dans Odoo

Description

Ce module implémente les notions d'EPCI et de commune dans Odoo :

  • De créer des EPCI
  • De créer des communes
  • De relier des communes à des EPCI

Installation

Lors de l'installation, ce module va ajouter trois menus sous le menu Configuration > Sociétés.
Les menus sont les suivants :




  • EPCIs : affiche la liste des EPCI
  • Communes : affiche la liste des communes
  • Hiérarchie des sociétés : affiche toutes les sociétés sous forme hiérarchique avec les communes dans les EPCI
