Mieux connaître vos contacts

Utiliser des informations personnelles pour enrichir vos relations

Description

Ce module introduit des nouvelles informations de contact

  • Information sur le lieu de naissance
  • Information sur l'âge et la date de naissance
  • Information sur le quotient familiale
  • Information sur le genre

Usage

Un nouvelle page contenant les informations personnelles se trouve dans la fiche d'un contact

Installation

Le paramétrage se fait dans le menu de configuration de la collectivité

Un nouveau groupe de droit d'accès est crée pour n'autoriser l'usage des informations personnelles qu'a une liste restreinte d'utilisateurs
