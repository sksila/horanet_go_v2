Gérez les supports d'identification et leurs usages

Description

Ce module ajoute les notions de badges et d'identifiants dans Odoo.

Usage

Un menu et ses sous-menus apparaissent dans l'onglet Collectivité

Un onglet sur la fiche d'un contact apparaît listant ses supports et tags.
