Ressources Javascript / CSS

Description

Ce module ajoute les ressources Javascript / CSS communes au modules frontoffice

  • Traductions du widget datepicker de jQuery
  • Des portlets
  • Un widget permettant de bloquer l'interface utilisateur
  • Un widget permettant de changer la taille du texte selon son conteneur
  • Un widget d'autocompletion
  • Bootstrap Toggle pour transformer des cases à cocher en bouton commutateur
  • SweetAlert pour afficher des messages dans des modales
  • Un fix pour obtenir les traductions javascript (toujours utile ?)
