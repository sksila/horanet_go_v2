Widgets et ressources pour le backend

Description

Ce module implémente les fonctionnalitées suivantes :

  • Avoir des formulaires plus larges en ajoutant l'attribut : o-form-sheet-width-wider sur la balise "sheet"
  • Possibilité pour un champ de se modifier après saisie et avant enregistrement (voir tool.force.dirty)
  • D'ajouter des bouttons (destinés à la redirection) dans les vues tree en spécifiant le widget : "redirect"
  • D'avoir une champ d'upload de fichier ayant une taille maximum de 250 Mo en spécifiant le widget : "bigfile_binary"
