Masques de saisie pour vos formulaire

Améliorez la façon dont sont saisies les données

Description

Basé sur le plugin jquery inputmask ( https://github.com/RobinHerbots/jquery.inputmask).
Un masque de saisie aide l'utilisateur à la saisie d'une donnée dans le bon format. C'est très utile pour les dates, les numéros de téléphone, ...

Usage

Ajoutez data-inputmask="masks" dans le champ. Par exemple :

  • <field name="email" data-inputmask="'alias': 'email'" />
  • <field name="ip_address" data-inputmask="'alias': 'ip'" />
  • <field name="phone" data-inputmask="'mask': '99-99-99-99-99'" />

Options disponibles :

  • 9: valeur numérique
  • a: valeur alphabétique
  • *: valeur alphabétique

Alias disponibles :

  • email
  • ip: adresse IPv4
  • url
