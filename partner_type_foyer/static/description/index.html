Regrouper vos contact en foyer

Utilisez les foyers pour créer des familles, collocation, groupes de personnes

Description

Ce module introduit un nouveau type de contact d'entreprise : le foyer. Il permet :

  • D'attacher un ou plusieurs foyers à un contact
  • De choisir des dates de début et fin de relation entre un contact et un foyer
  • De rechercher des foyers via le nom d'un de ses membres ou de l'adresse
  • À un partenaire de créer et gérer les membres de son foyer

Usage

Les partenaires de type foyer sont accessibles dans le nouveau menu "Citoyen/Foyer"

Les relations entre les partenaires et les foyers sont accessibles dans le menu de configuration pour les administrateurs
