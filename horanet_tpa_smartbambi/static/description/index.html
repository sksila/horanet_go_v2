TPA du système SmartBambi

Interface TPA entre Horanet GO et SmartBambi

Description

Ce module ajoute les fonctionnalités suivantes :

  • Descente des usagers depuis Horanet GO vers SmartIntegral - SmartBambi
  • Suivi des descentes des usagers
  • Fusion des usagers depuis Horanet GO et répercutée sur SmartIntegral - SmartBambi
  • Descente automatique des usagers s'inscrivant sur Horanet GO vers SmartIntegral - SmartBambi

Usage

Dans le menu Horanet GO -> Configuration -> Interfaces TPA

Descente des usagers depuis Horanet GO vers SmartIntegral - SmartBambi

La descente des usagers permet d'avoir une uniformité de données entre Horanet GO et SmartIntegral - SmartBambi. Toute modification d'informations de l'usager faite sur Horanet GO descendra dans le système SmartIntegral - SmartBambi, à condition biensûr que l'usager soit coché en tant que membre du TPA associé au métier visé de SmartIntegral.

Suivi des descentes des usagers

Une fois des modifications effectuées sur un usager synchronisé via le TPA, nous pouvons suivre la synchronisation pour savoir si elle a bien eu lieu et avoir des informations supplémentaires (horodatage de la synchronisation, message d'erreur, ...). Ces informations seront disponibles dans Horanet GO -> Configuration -> Interfaces TPA -> Synchronisations status.

Fusion des usagers depuis Horanet GO et répercutée sur SmartIntegral - SmartBambi

Afin de fusionner des usagers, il faut s'assurer que l'utilisateur ait bien les droits pour effectuer la fusion sur Horanet GO. Voir dans Configuration -> Utilisateurs.

Une fois que l'utilisateur a les droits nécessaires, dans la page d'un usager, il est possible de le déclarer comme étant un doublon et de démarrer la fusion.

Une fois la fusion Horanet GO terminée, une fusion dans SmartIntegral - SmartBambi sera demandée et pourra être suivie dans Horanet GO -> Configuration -> Interfaces TPA -> Synchronizations merge.

Descente automatique des usagers s'inscrivant sur le portail Horanet GO vers SmartIntegral - SmartBambi

Dans le cadre d'un portail métier, les usagers peuvent se créer un compte sur le portail Horanet GO et seront déscendus automatiquement vers SmartIntegral - SmartBambi. Il suffira de configurer le modèle d'utilisateur "Template Citoyen".

Pour configurer le modèle d'utilisateur "Template Citoyen", il faudra se rendre dans Configuration -> Utilisateurs et choisir le filtre "Utilisateurs inactifs".

Afin que les usagers du portail métier soient automatiquement synchronisés avec le TPA SmartBambi, il suffira de cocher la case "Synchronisation automatique avec SmartBambi" sur le modèle d'utilisateur "Template Citoyen".

Installation

Ce module doit être impérativement configurer pour pouvoir fonctionner correctement car sinon aucun échange ne sera fait avec le système SmartIntegral - SmartBambi.

Pour se faire, il suffit d'aller dans Horanet GO -> Configuration -> Configuration et de remplir les informations du bloc "Configurations TPA". Les informations à remplir pour la synchronisation des usagers seront les suivantes :

  • URL du service web SmartBambi
  • Méthode du service web appelée
  • Activation du TPA SmartBambi
