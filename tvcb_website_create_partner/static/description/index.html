Création d'usager depuis la caisse TVCB

Description

Ce module permet :

  • de créer un ou plusieurs usagers depuis la caisse TVCB
  • de créer un compte utilisateur à l'usager si choisi
  • de synchroniser automatiquement les nouveaux usagers dans TVCB

Installation

Il est nécessaire de configurer la caisse TVCB accèder à la page /tvcb/partner/create
