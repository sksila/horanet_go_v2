Module commun aux interfaces TPA

Contient les ressources nécessaires aux interfaces TPA

Description

Ce module ne devrait pas être installé directement, il sert de source aux différentes implementations de modules d'interface entre Odoo et les TPA (third party application). Il contient :

  • Un emplacement de paramétrage des services TPA
  • Une tâche planifiée d'appel de synchronisation
