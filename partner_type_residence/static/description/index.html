Améliorez la gestion des adresses

Utilisez des adresses précises pour mieux connaître vos clients

Description

Ce module introduit un système de gestion d'adresse par modèles. Il permet :

  • De recherche un Partner par son adresse
  • Valider la saisie des adresses pour s'assurer de toujours avoir des données saines
  • Grouper les Partners par composante d'adresse (personnes de la même ville/rue)
  • L'autocompletion lors de la saisie des addresse
  • Connaitre les numéro de rue d'une adresse

Usage

L'ensemble des modèles d'adresses sont accessibles dans le nouveau menu "Location/adresse"

Installation

Lors de l'installation ce module va convertir l'ensemble des adresses existante (sous forme de chaîne de caractère) en modèle d'adresses. Les numéros de rue seront extrait du champ "rue" si ils existent
