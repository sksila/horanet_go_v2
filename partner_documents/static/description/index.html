Améliorez la gestion des documents

Soumettez des documents en ligne et vérifiez leur statut

Description

Ce module permet d'afficher la liste des documents de l'utilisateur courant dans l'interface client (website). Il permet :

  • Définir des types de document
  • Assigner des dates d'expiration
  • Défini un état pour ses documents (archivé, périmé)
  • De soumettre des documents via le portail web
  • De les afficher
  • D'approuver ou non ces documents

Usage

Pour utiliser ce module, rendez-vous sur l'onglet "Documents" du menu principal.
