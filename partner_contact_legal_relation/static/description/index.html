Mieux connaître vos contacts

Renseignez les responsabilités légales entre vos contacts

Description

Ce module ajout une notion de responsabilité légale entre les contacts. Une personne peux être responsable d'une autre, c'est par exemple le cas d'une relation entre un enfant et ses parents

Usage

Un nouvelle page contenant les relations légales se trouve dans la fiche d'un contact
