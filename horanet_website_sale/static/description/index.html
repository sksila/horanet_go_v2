Portail de vente pour les collectivités

Adapte le portail de vente pour les collectivités

Description

Les adaptations apportées sont les suivantes :

  • La boutique peut être privatisée et être accessible par les utilisateurs connectés uniquement.
  • La case à cocher "J'accepte les conditions de vente", si elle est activée, est décochée de base.
  • Le formulaire de renseignement d'adresse dans le processus de panier a été enrichi avec les données du module better_address.

Usage

Configuration

La privatisation de la boutique et le cochage par défaut de la case à cocher des conditions de vente peuvent être configurés dans l'administration du site web.

Boutique privée

La boutique peut être privatisée et donc uniquement accessible pour les utilisateurs connectés. Il suffit de modifier le paramètre "La boutique est privée" dans les paramètres de l'administration du site web. Si un utilisateur non-connecté essaye d'accéder à la boutique, il sera redirigé vers la page de connexion

Acceptation des conditions de vente

Odoo propose de base une case à cocher pour valider la lecture des conditions de vente. Elle peut-être visible à la page de choix du type de paiement dans le workflow du panier. Elle peut être activée ou désactivée via le bouton "Personnaliser" lorsque l'on est connecté en tant qu'administrateur du site web.

Une fois activée, la case à cocher, normalement cochée de base, est maintenant cochée ou décochée suivant la configuration que vous avez entrée.

Renseignement des informations d'adresse

Il est nécessaire pour la facturation et la livraison d'avoir les informations d'adresse de l'utilisateur lorsqu'il effectue un achat. Le module collectivité amène un référentiel d'adresses, l'utilisateur doit donc retrouver son adresse au travers du référentiel installé.
