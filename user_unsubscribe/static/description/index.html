Désinscription utilisateurs

Permettez aux utilisateurs de se désinscrire

Description

Ce module permet de rajouter un lien "Désinscription" dans le pied de page du site web. Il gère les demandes dans l'onglet du même nom du menu Collectivité

Usage

Dans le menu Collectivité, les agents peuvent étudier les demandes de désinscription.
