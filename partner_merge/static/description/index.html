Fusionner vos contacts

Fusionner vos contacts pour éviter les doublons

Description

Ce module permet de fusionner les contacts

  • Fusion de 3 contacts maximum à la fois
  • Sélection du contact de destination
  • Possibilité de flagger un contact comme doublon

Usage

Deux nouveaux boutons se trouvent sur le formulaire d'un partenaire:

  • le premier affiche le nombre de contact marquérés comme doublon, un clic sur celui-ci affiche un assistant de fusion.
  • le second permet définir si un contact est un doublon ou non.
