Permet l'export des usagers n'ayant pas de support d'identification

Description

Ce module ajoute un assistant permettant d'exporter la liste des usagers n'ayant pas support d'identification actif.
L'export génère un fichier qui doit être envoyé à l'encarteur. L'encarteur imprime les cartes et modifie le fichier en y ajoutant les numéros CSN aux usagers correspondant.
Il nous renvoit alors ce fichier que l'on peut importer pour générer les assignations.

Usage

Un menu apparaît sous le menu Identification de l'onglet Collectivité
