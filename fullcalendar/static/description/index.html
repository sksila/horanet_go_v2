FullCalendar

Import FullCalendar jQuery library into Odoo module

Description

FullCalendar version 2.9.1

To get all documentation about this library, visit the FullCalendar site

Use

To use it, just declare an html div element as:

<div id="my_custom_calendar"></div>

And declare it at calendar widget container as:

<script type="text/javascript">
odoo.define('mymodule.myservice', function (require) {
    'use strict';

    var calendar = $('#my_custom_calendar');
    var params = {}; // my FullCalendar parameters
    calendar.fullCalendar(params);
});
</script>

Installation

Just have to install module and use it.
