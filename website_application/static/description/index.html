Téléservices

Module de base de gestion des téléservices

Description

Ce module ne doit pas être installé manuellement.

Il permet de créer et gérer les téléservices.
Les téléservices servent à faire des demandes sur le portail. Il doit être hérité pour ajouter des types téléservices.

Usage

Il vous suffit de gérer les téléservices dans le menu Téléservices de la collectivité. Seuls les membres du groupe "Agent" peuvent le faire.
